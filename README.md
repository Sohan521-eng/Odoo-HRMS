# 📌 Project Overview

Odoo-HRMS is a web-based Human Resource Management System that helps organizations manage employees, attendance, payroll, leave requests, reports, and HR operations through a modern dashboard.

# ✨ Features

- Employee Management
- Attendance Management
- Leave Management
- Payroll System
- AI HR Assistant
- Reports & Analytics
- Company Calendar
- Performance Dashboard
- Dark / Light Theme
- Responsive User Interface

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Sohan521-eng/Odoo-HRMS.git
```

Go to the project folder:

```bash
cd Odoo-HRMS
```

Install dependencies:

```bash
npm install
```

Run the backend:

```bash
python server.py
```
